# Security Policy

## Reporting a vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, use one of these private channels:

- [GitHub private vulnerability reporting](https://github.com/Locastic/Loggastic/security/advisories/new) (preferred)
- Email: paula@locastic.com

Please include a description of the issue, steps to reproduce, and the affected versions if
known.

## Supported versions

Security fixes are provided for the latest release line. Older versions
should upgrade to the latest release.
